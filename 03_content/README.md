# Content

Drafts, ideas, and work-in-progress for content that will appear on the site. Nothing here is published yet — published content lives in the actual site repo once the build exists.

## Structure

- **`essays/`** — written pieces, drafts, outlines
- **`videos/`** — video scripts, storyboards, concepts
- **`experiments/`** — tool/experiment ideas before they're built

## How to use

- Save any essay draft here as `.md` — frontmatter optional until you're ready to move it into the site
- Notes on video ideas or scripts go in `videos/`
- Small tool concepts or sketches go in `experiments/`

## Once the site is built

When an essay or video is ready to publish, it moves from this folder into the actual site's content directory (e.g. `content/essays/` inside the Astro project). This `03_content/` folder is a staging area for thinking and drafting, not for publishing.

## Light rule

If something sits here for more than 3 months without moving toward publication, it's probably not going to happen. Either kill it or actually work on it. Don't let this folder become a graveyard.
