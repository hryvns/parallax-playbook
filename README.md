# Parallax Playbook

**Name:** Parallax Playbook
- *Parallax* — the shift in perception that happens when you change viewpoints; the gap between how things look and how they actually are.
- *Playbook* — what you do about it. The record of your thinking, your methods, your honest work.

**What this is:** a personal site and creative workshop about trading, markets, and the psychology of both.
**Status:** specs written, build pending.

---

## How to navigate this repo

**If you're here to understand what the project is:**
1. Read `02_site/SITE_SPEC.md` first
2. Then `01_brand/brand_guidelines/MANIFESTO.md` for voice
3. Then `02_site/DESIGN_DIRECTION.md` for aesthetic

**If you're here to build the site:**
1. Start with `02_site/SITE_SPEC.md` (the what)
2. Then `02_site/DESIGN_DIRECTION.md` (the how it should feel)
3. Then `01_brand/` for design tokens (typography, color, voice)
4. Open Claude Code and start building

**If you're here to write/make content:**
1. Use `03_content/` as your staging area
2. Read `02_site/CONTENT_PLAN.md` for the format philosophy
3. Don't worry about a publishing schedule

## Principles carried forward

From the original manifesto and still true for the site:

- Serious, not scrappy
- Honest, not hype-y
- Beautifully made, with restraint
- Respect the reader
- Depth over reach
- Quality over schedule

When making any decision about the site or its content, these principles win.
